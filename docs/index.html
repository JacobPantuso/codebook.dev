Setup

Getting setup on codebook.dev is extremely simple. Follow these steps:

  1. Head to www.codebook.dev and enter your name to get started.
  2. Sign-in or create an account for free using the Netlify Authentication button. To answer that burning question. Yes, we now support GitHub O-Auth.
  3. After you've authenticated, you'll be redirected to the code webpage.
  4. Woooooosh! That was quick, you're all good to go.

Code Editor

codebook.dev uses a wide range of libraries to make coding online fun and easy for users. To begin coding, you'll need to create an account or sign-in to your existing account. Once you've done that, you'll be able to access the coding interface and start coding. Our editor is designed to highlight syntax in the selected language.

Code Compilation

codebook.dev uses a feature-rich API to generate the expected output from your code. We obtain the output using Judge0. This API is capable of compiling and executing code in over 50 programming languages. We plan to add more languages in the future.
